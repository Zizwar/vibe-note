## Getting Started

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/expo-55-starter.git
cd expo-55-starter

# Install dependencies
npm install

# Start development server
npm start
```

Then scan the QR code with **Expo Go** on your device, or press `a` for Android / `i` for iOS simulator.
