# خريطة الترقية السهلة — Vibe Note v1.5

> ترقية **سلسة درجة-درجة** تجعل التطبيق يبدو ويُحسّ مختلفاً عن v1 — **بلا هجرات قاعدة بيانات، بلا اختبارات، بلا ملفات كثيرة**. نشتغل بذكاء: نعيد استخدام ما هو موجود، ونعدّل بدل أن نضيف. الأشياء غير الواقعية (مزامنة سحابية، سوق مجتمعي، فرق) **مؤجّلة لـ v2**.

**المبادئ:**
- كل "درجة" (v1.5.x) قابلة للشحن وحدها وتُظهر فرقاً مرئياً.
- لا تغييرات في مخطط قاعدة البيانات (لا `ALTER`/هجرات) — نعمل ضمن الأعمدة والحقول الحالية (`variables`, `tags`, `content` JSON متاحة).
- تقليل الوزن: حذف كود/أصول ميتة بدل تكديس مكتبات.
- إعادة استخدام المكوّنات والأنماط القائمة (`PromptCard`, `useTheme`, `i18n`, `navigationStore`).

---

## درجات الترقية (بالترتيب)

### 🟢 v1.5.1 — طريقة عرض جديدة: "نوت/نصوص" (Notes view)
**الهدف:** شكل مختلف فوراً عن v1. حالياً 3 أوضاع (قائمة/شبكة/تصنيف) — نضيف **رابعاً** يعرض البرومبتات كملاحظات نصّية متراصّة (سطر عنوان + معاينة نص أطول، بلا بطاقات ثقيلة) — أخفّ بصرياً وأقرب لدفتر ملاحظات.
- **الملفات:** `HomeScreen.tsx` (إضافة `'notes'` لـ `ViewMode` + أيقونة)، `PromptCard.tsx` (نمط `variant="note"` خفيف) — **بلا ملفات جديدة**.
- **القبول:** زر رابع في شريط الأوضاع يعرض قائمة ملاحظات نظيفة، يدعم RTL.

### 🟢 v1.5.2 — إعادة تصميم شاشة الدردشة المستقلّة (Chat redesign)
**الهدف:** الشاشة الحالية شكلها لم يناسبك. نعيد تصميم تبويب الدردشة في `AIAssistantScreen`: فقاعات أنظف، رأس بهوية المساعد + المزوّد النشط، حالة فارغة بإيحاءات (chips اقتراحات سريعة)، صندوق إدخال أوضح، ومؤشّر "يكتب…".
- **الملفات:** `AIAssistantScreen.tsx` + مفاتيح `i18n` فقط.
- **القبول:** تبويب الدردشة يبدو حديثاً ومختلفاً، يعمل RTL، الاقتراحات السريعة تملأ الإدخال.

### 🟢 v1.5.3 — "دردش مع البرومبت" بدل النسخ (In-app run)
**الهدف:** بدل نسخ البرومبت لإعطائه لوكيل خارجي، تبدأ محادثة داخلية مهيّأة به. (الأساس موجود: زر "دردشة مع الذكاء" + `seedPrompt`.) نطوّره: عند الفتح يُرسَل البرومبت **مع رسالة سياق** ويبدأ ردّ المساعد تلقائياً (تشغيل فوري)، مع زر "أرسل كما هو".
- **الملفات:** `AIAssistantScreen.tsx`, `PromptDetailScreen.tsx`, `VariableFiller.tsx` (موجودة) + `i18n`.
- **القبول:** من البرومبت → دردشة جاهزة وتشغيل بنقرة، بلا خروج من التطبيق.

### 🟡 v1.5.4 — ربط البرومبتات (Prompt Chaining خفيف، بلا قاعدة بيانات)
**الهدف:** ربط برومبتات في "سلسلة" يُغذّي مخرج كلٍّ التالي — **دون تغيير المخطط**: السلسلة تُبنى وقت التشغيل في الدردشة (تختار برومبتاً تالياً من قائمة فيُضاف للمحادثة)، أو تُحفظ كقائمة معرّفات داخل حقل `variables` JSON الموجود (إعادة استخدام بلا عمود جديد).
- **الملفات:** `AIAssistantScreen.tsx` (منتقي "أضف برومبت للسلسلة")، `engine/` دالة مساعدة صغيرة، `i18n`.
- **القبول:** تشغيل سلسلة من ≥2 برومبت متتاليين في محادثة واحدة.

### 🟡 v1.5.5 — تخفيف الوزن وتلميع (Slim down & polish)
**الهدف:** التطبيق أخفّ وأنظف. حذف كود/أصول غير مستخدمة، توحيد الأنماط المكرّرة، إزالة استيرادات ميتة، وضبط README ليطابق الكود (السمات الفعلية).
- **الملفات:** عدّة، **بالحذف أكثر من الإضافة**. `README.md`, تنظيف `imports`/أنماط.
- **القبول:** حجم/تعقيد أقل، `tsc` و`expo-doctor` نظيفان، لا ميزات مكسورة.
