# Vibe Note - وصف متجر بلاي (العربية)

## اسم التطبيق
Vibe Note - مدير البرومبتات الذكي

## الوصف القصير (80 حرف كحد أقصى)
احفظ، نظّم واستخدم برومبتات الذكاء الاصطناعي بمتغيرات ديناميكية. مجاني وبدون إنترنت.

## الوصف الكامل

**Vibe Note** هو مكتبتك الذكية للبرومبتات — مصمم لكل من يستخدم أدوات الذكاء الاصطناعي يومياً.

توقف عن ضياع برومبتاتك بين الملاحظات والبوكماركات والمحادثات. Vibe Note يعطيك مكان واحد تحفظ فيه، تنظم، وتعيد استخدام كل برومبتاتك — لكل المنصات.

**المتغيرات الديناميكية**
أنشئ قوالب برومبت ذكية بمتغيرات:
- {{الموضوع}} — تعبئة أي قيمة وقت الاستخدام
- {{اللغة|العربية}} — قيم افتراضية جاهزة
- {{الأسلوب:رسمي|عادي|احترافي}} — قائمة اختيار
برومبت واحد، استخدامات لا نهائية. عبّي، انسخ، والصق — في ثوانٍ.

**تنظيم ذكي**
- صنّف حسب المجال: برمجة، كتابة، تسويق، تصميم، تعليم والمزيد
- حسب المنصة: ChatGPT، Claude، Gemini، Midjourney، DALL-E، Cursor، v0 والمزيد
- أضف وسوم مخصصة للتصفية السريعة
- ثبّت البرومبتات الأكثر استخداماً في الأعلى

**أدوات مدعومة بالذكاء الاصطناعي**
- دردش مع الذكاء حول هندسة البرومبتات
- استيراد ذكي: الصق نص خام والذكاء ينظمه في قالب
- إنشاء: صِف ما تحتاجه والذكاء يبني البرومبت
- تحليل: احصل على درجة جودة واقتراحات تحسين
- تعديل بالذكاء: حسّن أي برومبت بمساعدة AI

**سجل الاستخدام**
تابع كم مرة استخدمت كل برومبت. أعد استخدام القيم السابقة من السجل بنقرة واحدة.

**الخصوصية أولاً**
- كل البيانات مخزنة محلياً على جهازك
- بدون حساب، بدون تتبع، بدون إعلانات
- مفاتيح API مشفرة بتخزين آمن
- لا يتصل إلا بمزودات الذكاء التي تختارها (Gemini، OpenAI، OpenRouter)

**ميزات إضافية**
- الوضع الليلي
- دعم كامل للعربية والإنجليزية مع RTL
- استيراد وتصدير النسخ الاحتياطية (.vibe، .json)
- عداد التوكنات لكل برومبت
- مشاركة البرومبتات كملفات

**مجاني للأبد**
Vibe Note مجاني بالكامل بدون تكاليف مخفية، بدون اشتراكات، وبدون إعلانات.

مبني للمسوقين، المطورين، المصممين، صنّاع المحتوى، الطلاب — كل من يتحدث مع الذكاء الاصطناعي.

## ملاحظات الإصدار (ما الجديد)
- هوية جديدة: Vibe Note
- تحسين مساعد الذكاء مع تنقل أفضل
- تحسين تعبئة المتغيرات مع تحديث فوري
- سجل الاستخدام مع إعادة الاستخدام السريعة
- إضافة سياسة الخصوصية وشروط الاستخدام
- تحسينات في الأداء وإصلاح الأخطاء
