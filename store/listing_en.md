# Vibe Note - Play Store Listing (English)

## App Title
Vibe Note - Smart Prompt Manager

## Short Description (80 chars max)
Save, organize & use AI prompts with dynamic variables. Free & offline-first.

## Full Description

**Vibe Note** is the smart prompt library designed for anyone who uses AI tools daily.

Stop losing your best prompts in scattered notes, bookmarks, and chat histories. Vibe Note gives you one place to save, organize, and reuse all your AI prompts — across every platform.

**DYNAMIC VARIABLES**
Create reusable prompt templates with smart variables:
- {{topic}} — fill in any value at use time
- {{language|English}} — set default values
- {{tone:formal|casual|professional}} — dropdown selection
One prompt, infinite uses. Fill variables, copy, and paste — in seconds.

**SMART ORGANIZATION**
- Classify by category: Code, Writing, Marketing, Design, Education & more
- Tag by AI platform: ChatGPT, Claude, Gemini, Midjourney, DALL-E, Cursor, v0 & more
- Add custom tags for quick filtering
- Pin your most-used prompts to the top

**AI-POWERED TOOLS**
- Chat with AI about prompt engineering
- Smart Import: paste raw text, AI organizes it into a template
- Generate: describe what you need, AI creates the prompt
- Analyze: get a quality score and improvement suggestions
- Edit with AI: refine any prompt with AI assistance

**USAGE HISTORY**
Track how often you use each prompt. Reuse previous variable values from history with one tap.

**PRIVACY FIRST**
- All data stored locally on your device
- No account required, no tracking, no ads
- API keys encrypted with secure storage
- Only connects to AI providers you configure (Gemini, OpenAI, OpenRouter)

**MORE FEATURES**
- Dark mode
- Full Arabic & English support with RTL
- Import/export backups (.vibe, .json)
- Token counter for every prompt
- Share prompts as files

**FREE FOREVER**
Vibe Note is completely free with no hidden costs, no subscriptions, and no ads.

Built for marketers, developers, designers, content creators, students — anyone who talks to AI.
